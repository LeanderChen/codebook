# CodeBook

> 仓库用于记录个人及团队在`PHP`、`Javascript`、`Java`、`Python`等具体语言技术栈的学习、实施经验方法整理，此外也附有一些经典高效的`Snippet`供参考。笔者目标希望这本CodeBook能作为准备学习，或正在学习主流技术栈，并依托研发的朋友用以旁征博引参考的现实依据。
